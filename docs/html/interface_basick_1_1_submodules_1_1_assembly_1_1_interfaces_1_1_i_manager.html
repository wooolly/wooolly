T Basick.Submodules.Assembly.Interfaces.IManager.CreateCachedSingle< T > ( )

Creates an object from a single implementation of an interface type. If more than one implementation is found, an error should raise since the assembly manager cannot determine which one to instantiate. Way more efficient than CreateNonCachedSingle, as it will look in the type cache for the implementation type and it'll use the cached constructor during instantiation.
