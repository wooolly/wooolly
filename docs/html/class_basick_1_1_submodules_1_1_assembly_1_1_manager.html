◆ GetInterfaceCache< T >()

ICache Basick.Submodules.Assembly.Manager.GetInterfaceCache< T > ( )

Get a single implementation cache given an interface type. This only supports returning a single cache, so if there's multiple implementations, an exception should occur. These are convenience methods, because someone could access the interface cache directly by using the property getters.

Template Parameters
TThe interface to find a cache from.
Returns
The interface's implementation cache.
Exceptions
InvalidOperationExceptionIf the interface has multiple implementations, the system could not determine which cache to retrieve, so this error is raised.
KeyNotFoundExceptionThe interface does not have any implementations when this exception is raised.
