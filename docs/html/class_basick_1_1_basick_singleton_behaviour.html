Detailed Description

This special Basick Behaviour class implements the ISingleton interface and ensures that only a single instance ever exists of the descendant.

Template Parameters
TThe descendant type which must only ever exist once.

Executes in edit mode, meaning MultipleSingletonExceptions will throw immediately.

Exceptions
MultipleSingletonExceptionThis is thrown when multiple instances do exist.
